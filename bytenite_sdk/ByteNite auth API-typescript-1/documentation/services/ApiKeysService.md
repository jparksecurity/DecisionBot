## authApiKeys

Get all issued api keys for user

- HTTP Method: `GET`
- Endpoint: `/api_keys`

**Return Type**

`AuthApiKeysResponse`

**Example Usage Code Snippet**

```typescript
import { ByteNiteAuthApi } from 'bytenite auth api';

(async () => {
  const byteNiteAuthApi = new ByteNiteAuthApi({
    token: 'YOUR_TOKEN',
  });

  const { data } = await byteNiteAuthApi.apiKeys.authApiKeys();

  console.log(data);
})();
```
