To get started with the SDK, we recommend installing using `npm`:

```bash
npm install bytenite jobs api
```

## Authentication

### Access Token Authentication

The ByteNiteJobsApi API uses an Access Token for authentication.

This token must be provided to authenticate your requests to the API.

#### Setting the Access Token

When you initialize the SDK, you can set the access token as follows:

```ts
const sdk = new ByteNiteJobsApi({ token: 'YOUR_TOKEN' });
```

If you need to set or update the access token after initializing the SDK, you can use:

```ts
const sdk = new ByteNiteJobsApi();
sdk.token = 'YOUR_TOKEN';
```

## Setting a Custom Timeout

You can set a custom timeout for the SDK's HTTP requests as follows:

```ts
const byteNiteJobsApi = new ByteNiteJobsApi({ timeout: 10000 });
```

# Sample Usage

Below is a comprehensive example demonstrating how to authenticate and call a simple endpoint:

```ts
import { ByteNiteJobsApi } from 'bytenite jobs api';

(async () => {
  const byteNiteJobsApi = new ByteNiteJobsApi({
    token: 'YOUR_TOKEN',
  });

  const { data } = await byteNiteJobsApi.jobs.customerGetAll({
    paginationLimit: 5,
    paginationOffset: 5,
    orderBy: 'orderBy',
  });

  console.log(data);
})();
```
